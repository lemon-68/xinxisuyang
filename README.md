1. 对信息的获取、加工、管理、表达和交流的能力；
2. 对信息及信息活动的过程、方法、结果进行评价的能力；
3. 发表观点、交流思想、开展合作并解决学习和生活中实际问题的能力；
4. 遵守相关的伦理道德和法律法规。
